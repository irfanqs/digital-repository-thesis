Welcome to SU ETD Repository
Students can register, log in, and eventually submit their thesis PDFs for library review and publication.

This is a prototype connected to the Spring Boot backend. Right now:
• Register → POST /api/auth/register
• Login (student) → HTTP Basic, checking GET /api/theses/mine
• Browse → UI only (search API to be added later)

Student Login
Sign in with your SU email and password. For now, this login is for student accounts.

Don't have an account? Register here

Student Registration
Create a new student account for thesis submission. Faculty and major are used to label your program.
Password must be at least 14 characters and include: uppercase, lowercase, number, and special character.

Already have an account? Login here
